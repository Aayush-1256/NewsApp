# NewsApp (AppNews)
An ANDROID APP(Kotlin) which shows REALTIME NEWS CARDS based on different categories like BUSINESS , HEALTH , SPORTS etc. Also gives the function to SEARCH NEWS. All Done using the NEWSAPI.
